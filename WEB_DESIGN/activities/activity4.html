There are different types of number systems, such as:

  • Decimal
  • Binary
  • Octal
  • Hexadecimal

Each number system has its own base and digits.

For example, the decimal number system has a base of 10 and digits from 0 to 9.

Binary has a base of 2 and digits 0 and 1.

Octal has a base of 8 and digits from 0 to 7.

Hexadecimal has a base of 16 and digits from 0 to 9 and A to F.

Decimal Number System

Decimal is the most commonly used number system.

It has a base of 10 and digits from 0 to 9.

Binary Number System

Binary is used in computers and has a base of 2.

It uses only two digits, 0 and 1.

For example, the number 1010 in decimal is:

1 * 2^3 + 0 * 2^2 + 1 * 2^1 + 0 * 2^0 = 10
